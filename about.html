About MyLibrary

Project overview and purpose

Project Purpose

MyLibrary is a digital library website that allows users to browse books, access external information, save favourites, and share reviews.

User Requirements

  • View available books
  • Access external info via Wikipedia
  • Save favourites for later
  • Easy navigation on all devices

Technologies Used

  • HTML5
  • CSS3
  • JavaScript (localStorage)
  • External APIs (Wikipedia links)
